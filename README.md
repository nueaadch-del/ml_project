# ml_project


1. 프로젝트 제목 - 뇌졸중 발생 예측 모델

2. 프로젝트 주제 - 건강 데이터를 바탕으로 발생 가능성 예측하는 것

3. 데이터셋 설명 - 나이, 평균 혈당 수치, 고혈압, BMI, 흡연, 직 등의 정보가 있다

4. 사용한모델 - XGBoost모델 사용

5. 주요 결과 - 나이가 뇌졸중 발생 예측에 가장 큰 영향을 주는 변수로 나타났다

6. 결론 - 이 프로젝트를 통해 건강 데이터를 활용하면 뇌졸중 발생 가능성을 예측할 수 있다는 것을 알게 되었다. 또한 나이, BMI, 혈당 같은 건강 데이터가 예측에 중요하다는 것을 알게 되었다

7. 개선 방향 - 데이터 양을 더 늘리고 더 다양한 건강 데이터들을 추가하면 더 좋을 것 같다

8. 구조 설명 - 뇌졸중 관련 건강 데이터를 불러오고 필요한 데이터 정리를 했다. 그리고 나서 XGBoost모델를 이용하여 뇌졸중 발생 여부를 예측하였다. 마지막으로 모델 결과를 그래프를 통해 확인을 하였
