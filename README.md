# ml_project


1. 프로젝트 제목 - 뇌졸중 환자 발생 예측 모델 만들기
 
2. 프로젝트 주제 - 건강 데이터를 바탕으로 발생 가능성 예측하는 것

3. 데이터셋 설명 - 뇌졸중 발생 여부 관련딘 건강 정보가 포함되어 있다.
  나이, 고혈압 여부, 평균 혈당 수치, BMI, 흡연 상태, 직업, 뇌졸중 발생 여부
   BMI 결측치: BMI 평균값으로 대체하였다. / age 결측치: 테스트 데이터에서 결측치가 있는 행을 제거하였다.

4. 사용한모델 - Logistic Regression, KNN, Decision Tree, Random Forest, AdaBoost, XGBoost
최종적으로는 XGBoost 모델을 사용하였

5. 주요 결과 - 여러 모델을 학습한 뒤 하이퍼파라미터를 조정하고, 데이터 불균형을 줄이기 위해 balanced를 적용하였다.
Feature Importance를 확인한 결과, 뇌졸중 예측에 가장 큰 영향을 준 변수는 age 였다.  
Threshold 조정에서는 큰 성능 향상을 찾기 어려웠고, 최종적으로 ROC-AUC Curve를 통해 모델의 분류 성능을 확인하였다.

6. 결론 - 이 데이터셋은 뇌졸중 환자보다 뇌졸중이 아닌 사람의 비율이 훨씬 높은 불균형 데이터였다.
그래서 모든 사람을 “뇌졸중이 아니다”라고 예측해도 정확도가 높게 나올 수 있다.  
하지만 이런 결과는 실제 뇌졸중 환자를 제대로 찾았다는 뜻은 아니기 때문에 좋은 모델이라고 보기 어렵다.
뇌졸중 예측에서는 환자를 놓치는 것이 더 위험하다고 생각해서 Accuracy뿐만 아니라 Recall과 F1-score도 함께 확인하였다. 여러 결과를 비교해 본 결과, XGBoost 모델이 가장 괜찮은 성능을 보여 최종 모델로 선택하였다.

7. 개선 방향 - 현재 데이터셋은 전체 데이터 양이 많지 않고, 뇌졸중 환자 데이터 비율도 낮다는 한계가 있었다.앞으로는 다른 뇌졸중 관련 데이터셋을 추가로 가져와 기존 데이터와 병합하면 더 좋은 결과를 얻을 수 있을 것이다. 또한 혈압, 가족력, 운동량 같은 건강 정보를 더 추가하거나, 딥러닝 모델을 사용해 성능을 비교해 볼 수 있다.
